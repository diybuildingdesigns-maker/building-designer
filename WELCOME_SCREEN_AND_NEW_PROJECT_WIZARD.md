# DIY Building Designs Professional
# WELCOME_SCREEN_AND_NEW_PROJECT_WIZARD.md

## Purpose

Define the first experience every user has when launching the application.

The goal is to remove the intimidation of a blank CAD screen and guide users into creating a building within minutes.

---

# Welcome Screen

The application opens with a clean dashboard.

Options:

- New Metal Building
- Open Existing Project
- Recent Projects
- Learning Center
- Settings

The center of the screen should ask one question:

"What would you like to design today?"

---

# New Project Wizard

The wizard should require only the information needed to create an initial building.

## Step 1 – Project

- Project Name
- Customer
- Job Location

## Step 2 – Building

- Width
- Length
- Eave Height
- Roof Pitch
- Number of Bays

As values are entered, a live preview updates.

## Step 3 – Finish

When the user clicks Create Building, the software automatically:

- Creates the Building object
- Generates primary frames
- Generates bays
- Generates secondary framing
- Opens the main workspace

The user should never see an empty drawing area.

---

# Guiding Principles

- Ask questions instead of exposing tools.
- Create a usable building immediately.
- Allow experienced users to skip the wizard.
- Remember previous defaults when appropriate.

---

# Future Vision

Eventually the wizard may include templates such as:

- Warehouse
- Garage
- Shop
- Agricultural Building
- Commercial Building
- Custom

Templates should simply pre-fill values; users remain free to modify everything afterward.
